4. JavaScript 비교 연산자 심화

== (동등 연산자)

값만 비교
타입 변환 후 비교
예: "5" == 5 는 true


=== (일치 연산자)

값과 데이터 타입 모두 비교
엄격한 비교
예: "5" === 5 는 false
보다 안전한 코딩을 위해 권장되는 방식
