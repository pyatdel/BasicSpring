1. 컴퓨터 시스템 개발 구조


하드웨어 설계: 전자회로(표준) 기반
시스템 개발자/엔지니어: OS 개발 담당
소프트웨어 개발자: 하드웨어 발전 속도를 고려하여 학습 방향 설정 필요


2. localhost (127.0.0.1)의 목적


서버 개발 시 단일 PC에서 테스트 가능
클라이언트 프로그램을 자신의 IP로 루프백하여 서버-클라이언트 환경 시뮬레이션


3. 데이터 구조 표기


[] : 배열 형태로 데이터 수신
{} : VO(Value Object) 또는 Map 형태로 데이터 수신


4. 가상화 도구들


Virtual Box

오라클 제작의 무료 가상머신 프로그램
한 컴퓨터에서 다양한 OS 환경 구축 가능


Docker: Ubuntu 지원을 위해 개발
WSL(Windows Subsystem for Linux): Windows에서 Linux 실행


5. Linux 주요 개념


pwd: 현재 작업 디렉토리 출력
중요 리눅스 명령어:

redirect (>): 출력 방향 변경
& (background): 백그라운드 프로세스 실행
| (pipeline): 명령어 연결
sudo: 관리자 권한으로 명령 실행




6. 실행 경로의 차이


일반 명령어: 환경변수($PATH) 기준으로 검색
./실행파일: 현재 디렉토리 기준으로 검색

i: 입력 모드
ESC: 명령 모드
